登录错误状态合集

本页集中展示登录流程中可能遇到的各种异常状态。每个状态都有清晰的提示与下一步操作建议。

未安装钱包
E01 · NoWallet

未检测到 Web3 钱包

你的浏览器中尚未安装钱包扩展。请安装 MetaMask 或使用扫码登录。

error.code = WALLET_NOT_DETECTEDFE-1100
用户拒绝授权
E02 · UserRejected

你在钱包中点击了"拒绝"

登录已取消。如需登录 FansLoop，请重新发起连接并在钱包中点击"连接"。

error.code = 4001 (USER_REJECTED)FE-1201
钱包网络不匹配
E03 · WrongChain

当前网络不支持

你的钱包当前位于 BSC 网络。FansLoop 仅支持 Ethereum / Polygon / Base 等 EVM 主网。

chainId = 56 (BSC)expected = 1, 137, 8453
签名请求超时
E04 · Timeout

签名请求已超时

你在 30 秒内未完成签名。这可能是钱包窗口被遮挡。请重新发起登录并尽快完成签名。

error.code = SIGN_TIMEOUTelapsed = 30s
签名验证失败
E05 · InvalidSig

签名校验失败

服务器无法验证你的签名。这可能是 Nonce 已失效或签名内容被篡改。请重新登录。

error.code = INVALID_SIGNATUREBE-2305
账号已被限制
E06 · Banned

该地址已被封禁

因违反社区规范，该钱包地址 (0x7A3F...3F2C) 已被永久封禁。如有异议请申诉。

error.code = ADDRESS_BANNEDbanned_at = 2026-03-12
服务繁忙
E07 · ServerBusy

服务暂时繁忙

登录服务正在处理大量请求。请稍后再试，或查看实时状态页。

HTTP 503retry-after = 30s
WalletConnect 二维码失效
E08 · QRExpired

二维码已过期

WalletConnect 会话已超过 5 分钟未连接。请刷新二维码后重新扫码。

session.expired = trueWC-Relay
